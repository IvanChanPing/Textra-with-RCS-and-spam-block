WHAT THIS FOLDER IS
-------------------
An integration hand-off bundle. It contains two pieces of working
software plus the instructions to wire them into an Android
messaging app's SOURCE CODE — a normal Gradle / Android Studio
project, NOT a decompiled or smali-patched APK:

  1. messaging-engine-rust/  — a Rust port of the Google Messages "libgm"
     web protocol library (the client half of the open-source
     mautrix-gmessages bridge). It speaks the Google Messages
     pairing + send + receive protocol and is exposed to Kotlin /
     Java through UniFFI bindings.

  2. A rounded-corner parallax Activity transition — the iOS /
     OnePlus look where a screen slides in with rounded leading
     corners while the screen behind it dims. Built entirely with
     standard Android APIs.
